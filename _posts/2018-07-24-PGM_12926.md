## [1] Answer Code (18. 07. 24)

``` cpp

string solution(string s, int n) {
    string answer = "";
   
    for(int i=0; i<s.length(); i++){
        if (s[i] >= 97 && s[i] <= 122) { // Lower Case
            s[i] = s[i] + n;
            if( s[i] > 122){
                s[i] -= 26;
            }
        }
        else if (s[i] >= 65 && s[i] <= 90) { // Upper Case
            s[i] = s[i] + n;
            if( s[i] > 90){
                s[i] -= 26;
            }
        }
    }
    answer = s ;

    return answer;
}


```
